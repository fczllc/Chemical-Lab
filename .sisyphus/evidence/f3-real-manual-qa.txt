F3 Blocker Fix: PASS
Date: 2026-05-14

Diagnosis:
- Reproduced the blocker with `npx playwright test tests/ui/shared-game-feedback-overlay.spec.ts`: default workers reported 5 passed / 1 failed, with the helper overlay test receiving computed `pointer-events` as an empty string instead of `none`.
- Inspected `tests/ui/shared-game-feedback-overlay.spec.ts`, `src/modules/gameFeedbackOverlay.js`, `src/styles/games.css`, `index.html`, and notepad history. The implementation CSS rule exists at `.game-rule-feedback-overlay { pointer-events: none; }`, and the module preserves one body-mounted overlay with `aria-hidden="true"` and auto-removal.
- Root cause: test race against a transient auto-removing overlay. The old helper did separate locator assertions for count/text/class before polling `locator.evaluate(getComputedStyle(...))`, so under default parallel load the sampled locator could reach a disconnected/removed style state and return an empty computed value.

Change:
- Modified only `tests/ui/shared-game-feedback-overlay.spec.ts`.
- Reworked `expectOverlay` to use one state-based `expect.poll(() => page.evaluate(...))` snapshot that freshly queries the current DOM and atomically asserts count 1, exact emoji text, `is-visible`, `aria-hidden="true"`, and computed `pointer-events: none`.
- Did not remove or weaken the pointer-events assertion; no source/game rule/scoring/timer/quiz/collector behavior was changed.

Verification:
- `lsp_diagnostics` on `tests/ui/shared-game-feedback-overlay.spec.ts`: no diagnostics.
- `npx playwright test tests/ui/shared-game-feedback-overlay.spec.ts`: PASS, 6 passed using default workers, completed in 38.1s.
- `npm run build`: PASS, Vite production build completed in 5.88s; only existing chunk-size warning was emitted.

Research notes:
- Playwright guidance from background research: for transient/disappearing elements, prefer `expect.poll` with `page.evaluate` and DOM lookup inside the browser tick to avoid stale locator/element-handle races.
- Local overlay usage search found no additional overlay call sites beyond drag, memory, reaction, and full quiz; quick quiz and collector remain excluded.
